3. Как изменить информацию (имена, роли, био)
Если вы хотите изменить не только картинку, но и имя или описание админа, вам нужно отредактировать файл: components/ChatFeed.tsx

Найдите там блок const ADMINS, который выглядит так (примерно 23-я строка):

typescript
const ADMINS = {
    novokosino: [
        { 
          id: 'n_ivan', 
          name: 'Иван', 
          role: 'Senior Admin', 
          bio: 'Любит шутеры и энергетики...', 
          avatar: 'ape' // Здесь можно указать любое имя файла без .png
        },
        // ... остальные админы
    ],
    // ...
}
Итого:
Загрузите картинку в public/images/social-hub/.
Назовите её, например, ivan.png.
В файле components/ChatFeed.tsx в поле avatar напишите 'ivan'.
