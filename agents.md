Ты — Senior Frontend Engineer & UX Designer, специализирующийся на десктопных веб-приложениях. Задача: Спроектировать и реализовать код для "Первого Экрана" (Onboarding) платформы CyberX. Контекст: Пользователь открывает сайт с ПК. У нас есть секунды, чтобы вовлечь его. Экран должен быть визуально богатым, использовать пространство монитора, поддерживать управление мышью (ховеры) и клавиатурой (Enter для подтверждения).

Подключи следующие скиллы для реализации:

@ui-ux-pro-max

Зачем: Для построения Visual Hierarchy на широком экране. На десктопе много пустого места; используй этот скилл, чтобы сфокусировать внимание строго на центре (сетка аватаров), затемнив или размыв фон (backdrop blur). Также используй правила типографики для крупных заголовков.

@marketing-psychology

Зачем: Реализация Endowed Progress Effect. Визуализируй, что выбор аватара — это уже 50% успеха. Используй принцип Investment: выбрав персонажа, пользователь психологически "вкладывается" и с меньшей вероятностью закроет вкладку на этапе ввода ника.

@onboarding-cro

Зачем: Примени паттерн Gradual Engagement. Не показывай сразу форму регистрации. Сначала — фановая часть (выбор аватара), и только после клика — плавный переход (Transition) к полю ввода.

@form-cro

Зачем: Оптимизация ввода Никнейма.

Desktop Spec: Автофокус (autoFocus) на поле ввода сразу после появления. Поддержка отправки по Enter. Валидация в реальном времени (без лишних кликов "Проверить").

@react-best-practices

Зачем: Для мгновенного отклика UI. Состояние выбора аватара и ввода ника должно храниться в локальном стейте (или Zustand/Context) и синхронизироваться с localStorage без задержек. Реализуй hover-эффекты для мыши (скейл аватара при наведении), которые недоступны на мобилке.

Технические требования к Десктопу:

Сетка аватаров: Горизонтальная или плиточная (например, 2 строки по 4 аватара), центрированная.

Interactions: Явные Hover-эффекты (подсветка, неоновый бордер, легкое увеличение) для подсказки кликабельности.

Navigation: Пользователь должен иметь возможность выбрать аватар кликом мыши, мгновенно увидеть поле ввода, вбить ник и нажать Enter — всё это за 5-7 секунд.
